== Migração de dados

O serviço Red Hat OpenShift na AWS, bem como o Amazon FSx for NetApp ONTAP (FSx ONTAP), fazem parte do portfólio de serviços da AWS.  O FSx ONTAP está disponível em opções de AZ única ou AZ múltipla.  A opção Multi-Az fornece proteção de dados contra falhas na zona de disponibilidade.  O FSx ONTAP pode ser integrado ao Trident para fornecer armazenamento persistente para aplicativos em clusters ROSA.

A migração de aplicativos de contêiner envolve:

* Volumes persistentes: isso pode ser feito usando o BlueXP.  Outra opção é usar o Trident Protect para lidar com migrações de aplicativos de contêiner do ambiente local para o ambiente de nuvem.  A automação pode ser usada para o mesmo propósito.
* Metadados do aplicativo: isso pode ser feito usando o OpenShift GitOps (Argo CD).
